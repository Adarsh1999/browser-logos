## How to get this logo

* Install with:

  * [`npm`](https://www.npmjs.com/): `npm install --save-dev @browser-logos/uc-mini`
  * [`Yarn`](https://yarnpkg.com/): `yarn add --dev @browser-logos/uc-mini`

* Use [`cdnjs`](https://cdnjs.com/libraries/browser-logos)
